## ✨ Key Features

- 🚀 **Ultra Fast Performance** – Powered by **React** & **Tailwind CSS**
- 🎨 **Stunning UI/UX** – Minimal, modern, and highly interactive
- 🎥 **Animated Transitions** – Built with **Framer Motion**
- 🧭 **Carousel Integration** – Sleek sliders using **SwiperJS**
- 🧩 **Fully Responsive** – Looks perfect on mobile, tablet, and desktop
- 🔍 **SEO Optimized** – 100% Lighthouse SEO score ⚡
- 🎯 **Clear CTAs** – Buttons like “Start Free Trial” and “How it Works”
- 📞 **Live WhatsApp Chat** – Floating button for direct contact
- 🛡️ **Beautiful Icons** – Courtesy of **Heroicons**

## 🚀 Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/home-of-languages.git

# Navigate to the project folder
cd home-of-languages

# Install dependencies
npm install

# Run the development server
npm run dev
